# Git Practice 🔧

Practicing Git fundamentals as part of my Linux/Cloud Engineer learning journey.

## What I've practiced so far

### Day 1 — Git Basics
- `git init` — initialize a repository
- `git config` — set up username & email
- `git add` — stage changes
- `git commit` — save snapshots with messages
- `git log` — view commit history
- `git remote add` — connect local repo to GitHub
- `git push` — upload commits to GitHub

### Day 2 — Branching & Merging
- `git branch` — created a new branch (`feature-update`)
- `git checkout` — switched between branches
- Made changes on the feature branch and committed them separately
- `git merge` — merged the feature branch back into the main branch
- Faced and resolved a real "divergent branches" push conflict using:
  - `git config pull.rebase false`
  - `git pull --allow-unrelated-histories`
